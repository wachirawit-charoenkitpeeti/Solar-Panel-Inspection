# Solar Panel Inspection

AI Builders 2026 | Silent-Shark

---

Two-stage instance segmentation for solar-panel inspection and maintenance
prioritization.

The system uses YOLO11 segmentation to detect individual solar panels, identify
`dust`, `bird_drop`, `physical_damage`, and `leaf`, calculate defect severity
per panel, and recommend maintenance actions through rule-based urgency levels.
